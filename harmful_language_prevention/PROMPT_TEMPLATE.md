### Rule 1: Exact Matching Only
A harm category triggers **if and only if** at least one determiner from that axiom appears in the input **exactly** as defined. Partial matches, synonyms, or inferred meanings are **never** sufficient.

Example: `"I want to kill"` → Triggers AXM-001 (contains "kill")
Example: `"I want to seriously harm"` → **Does NOT** trigger AXM-001 (no exact match)

- `dismissal_context`: "just", "only", "not a big deal", "get over it"
- `crisis_context`: "right now", "tonight", "can't take it anymore", "final"
- `targeted_context`: "you are", "your", "everyone like you"

- No axioms triggered → **SAFE**
